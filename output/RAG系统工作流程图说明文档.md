### 2.2 关键代码对应说明

| 流程步骤  | 代码位置                             | 功能说明           |
| --------- | ------------------------------------ | ------------------ |
| 文件上传  | `UploadController`                 | 接收分片文件       |
| MinIO存储 | `MinioClient`                      | 保存原始文件       |
| Kafka消息 | `FileProcessingTask`               | 异步处理任务       |
| 文档解析  | `ParseService.parseAndSave()`      | Tika解析+HanLP分词 |
| 向量化    | `VectorizationService.vectorize()` | 调用Embedding API  |
| ES索引    | `ElasticsearchService.bulkIndex()` | 批量存储向量       |
